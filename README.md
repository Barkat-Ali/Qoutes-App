# ✨ Quote Submission App ✨

A simple and clean web application to **Submit, Approve, and Display Quotes**  
Built with:

- ⚙️ **Node.js + Express**
- 🗄️ **MongoDB (Mongoose)**
- 🎨 **HTML, CSS, JavaScript**

---

## 🎯 Core Features

- 📖 View approved quotes (Public Page)
- 📝 Submit new quotes (Public Form)
- 🔐 Admin dashboard to approve/delete quotes
